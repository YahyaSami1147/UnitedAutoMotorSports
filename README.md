# MotoShowroom Frontend

## Pages
- `index.html` — Home
- `catalog.html` — Bike catalog
- `about.html` — About us
- `contact.html` — Contact page
- `bike-detail.html` — Bike detail page
- `admin/login.html` — Admin login
- `admin/dashboard.html` — Admin dashboard
- `admin/add-bike.html` — Add new bike form
- `admin/manage-bikes.html` — Manage bikes view
- `admin/bookings.html` — Bookings management
 
- `system-1.html` — Management system UI 1
- `system-2.html` — Management system UI 2

## Usage
Open `index.html` in your browser or serve the folder using a static server.

## Notes
The site uses Tailwind CDN for styling and a small shared script for mobile navigation and gallery interactions.
